주요 기능

필요한 기능만 직접 켜세요

알뜰이의 모든 기능은 처음에 꺼져 있으며, 확장 팝업에서 원하는 기능만 선택할 수 있습니다.

01

정렬

검색 결과를 단위가격, 할인율, 가격 기준으로 다시 살펴볼 수 있게 돕습니다.

02

필터

직접 지정한 키워드를 기준으로 보고 싶지 않은 결과를 화면에서 가릴 수 있습니다.

03

페이지 표시 항목 조정

검색 페이지에서 필요하지 않은 표시 요소를 선택해 화면을 간결하게 정리합니다.

04

장바구니 바로 담기

검색 화면에 제공되는 담기 동작을 더 가까운 위치에서 실행할 수 있도록 보조합니다.

비공식 도구

쿠팡과 제휴하지 않았습니다

알뜰이는 개인이 비영리로 배포하는 오픈소스 도구이며, 쿠팡 및 그 운영사와 제휴·후원·위탁 관계가 없습니다. 제3자 서비스의 화면이나 정책 변경에 따라 기능이 달라질 수 있습니다.

데이터 고지

설정과 의견을 구분해 다룹니다

기능 설정은 이용자의 브라우저 저장소에 보관됩니다. 아래에 자발적으로 제출한 의견 문구는 공개 목록에 표시되고 서버에 저장되며, 웹 양식에서는 확장 버전을 보내지 않습니다.
