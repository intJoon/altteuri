# 출처

이 문서는 알뜰이 1.16.5의 외부 자료, 제3자 코드·라이선스, 실제 페이지 관찰 근거를 기록한다. 일회성 실행 로그, CDP 좌표, 폐기된 DOM 셀렉터 목록은 포함하지 않는다.

## 공식 플랫폼 문서

### Chrome Extensions

- Manifest V3 개요: https://developer.chrome.com/docs/extensions/develop/migrate/what-is-mv3
- `chrome.storage`: https://developer.chrome.com/docs/extensions/reference/api/storage
- 콘텐츠 스크립트: https://developer.chrome.com/docs/extensions/develop/concepts/content-scripts
- `chrome.scripting`: https://developer.chrome.com/docs/extensions/reference/api/scripting
- 원격 호스팅 코드 정책: https://developer.chrome.com/docs/extensions/develop/migrate/remote-hosted-code

적용 범위:

- 설정은 `chrome.storage.sync`와 `chrome.storage.local`에 저장한다.
- 쿠팡 페이지 기능은 Manifest V3 콘텐츠 스크립트로 실행한다.
- 장바구니 응답 관찰은 번들된 `page-cart-hook.js`를 `chrome.scripting.executeScript`로 MAIN world에 주입한다.
- 확장 실행 코드는 저장소에 포함하며 원격 JavaScript를 불러오지 않는다.

### Vercel과 Neon

- Vercel Functions: https://vercel.com/docs/functions
- Vercel Privacy Notice: https://vercel.com/legal/privacy-notice
- Neon serverless driver: https://neon.com/docs/serverless/serverless-driver
- Neon Privacy Policy: https://neon.tech/privacy-policy
- Neon Data Processing Addendum: https://neon.tech/dpa

적용 범위:

- `web/api/comments.mjs`는 Vercel 서버리스 API로 실행된다.
- `web/lib/db.mjs`는 Neon serverless driver로 PostgreSQL에 연결한다.
- 의견 처리의 위탁·국외 이전 세부 사항은 [`개인정보처리방침.md`](개인정보처리방침.md)에 기재한다.

## 쿠팡 페이지 관찰 근거

알뜰이는 쿠팡과 제휴하지 않았으며, 아래 내용은 공개 페이지 또는 사용자가 제공한 마크업을 기능 호환성 확인 목적으로 관찰한 결과다. 쿠팡의 공식 API 계약이 아니므로 언제든 변경될 수 있다.

### 검색 결과

- 상품 목록은 `#product-list`, 상품 카드는 `ProductUnit_productUnit` 계열, 상품명은 `ProductUnit_productName` 계열을 기준으로 찾는다.
- 2026-07-12 관찰에서 판매가·할인율·단위가의 안정적인 의미 클래스가 사라진 사례가 확인되어, 현행 구현은 `12,900원`, `13%`, `(100g당 516원)` 같은 텍스트 패턴을 주 기준으로 사용한다.
- 검색어는 `/np/search`의 `q`, 검색 개수는 `listSize`를 사용한다.
- 키워드 입력 영역은 쿠팡 필터 영역을 삽입 기준으로 사용하고, DOM이 바뀌면 자체 UI를 다시 구성한다.

### 표시 항목 프리셋

다음 항목은 2026-07-11~16의 실제 페이지 관찰 또는 사용자 제공 HTML을 바탕으로 `preset-data.js`에 반영했다.

- 검색 결과: 상단 배너, 브랜드 배너, 맞춤 상품 광고, 그리드 광고 상품, 특가·추천 캐러셀
- 상품 상세: 비교·추천·프로모션·브랜드 관련 캐러셀
- 장바구니: 하단 광고·함께 구매 추천
- 주문목록: 우측 프로모션 배너·하단 프로모션 슬라이더

프리셋의 단일 진실 공급원은 `preset-data.js`다. 셀렉터와 현재 항목명은 해당 파일을 우선하며, 이 문서에 과거 좌표나 폐기 셀렉터를 복제하지 않는다.

### 장바구니 바로 담기

2026-07-17 실제 UI에서 다음 성공 신호를 확인했다.

- `담기 완료`
- `상품이 장바구니에 담겼습니다`
- `장바구니 보기`, `장바구니 가기`, `장바구니 바로가기`
- 장바구니 관련 fetch/XHR의 성공 응답

헤더 장바구니 숫자는 미리보기 내용과 동기화되지 않는 사례가 있어 성공 근거에서 제외했다. 타임아웃도 성공으로 추정하지 않는다.

## 유지보수 기준

- 외부 문서의 URL·정책·라이선스는 의존성이나 법적 고지를 변경할 때 다시 확인한다.
- 쿠팡 DOM 관찰은 날짜와 대상 페이지를 남기되, 재현에 필요하지 않은 좌표·브라우저 세션·실행 로그는 커밋하지 않는다.
- 외부 코드를 추가하거나 이식하면 실제 사용 여부, 버전, 저작권자, 라이선스 전문 또는 배포에 필요한 고지를 함께 갱신한다.
