# 개인정보처리방침

> 본 문서는 대한민국 법령에 근거한 법적 고지로, 예외적으로 한국어로 작성한다.
> 「쿠팡 검색결과 애드온」(이하 "본 확장")의 개인정보 처리에 관한 방침입니다.
> 시행일: 2026-07-13

## 1. 총칙 — 개인정보 수집 여부

본 확장은 개인이 비영리로 배포하는 오픈소스 브라우저 확장 프로그램으로, **이용자의 개인정보를 수집·전송·저장하지 않습니다.** 본 확장의 개발자는 이용자의 데이터를 전송받는 어떠한 서버도 운영하지 않으며, 이용자의 브라우징 기록·계정 정보·구매 내역 등 개인정보에 접근하지 않습니다.

본 확장은 기능 동작에 필요한 **설정값(환경설정)만을 이용자 본인의 브라우저 저장소(`chrome.storage`)에 저장**합니다. 이 값은 외부로 전송되지 않습니다.

## 2. 처리하는 정보 항목 및 목적

본 확장이 브라우저 저장소에 저장하는 값은 다음과 같으며, 모두 **확장 기능의 설정 유지**만을 목적으로 합니다.

| 구분 | 항목(예) | 목적 |
|------|----------|------|
| 기능 on/off | 애드온 사용 여부, 정렬·필터·광고요소 제거·검색개수 고정 사용 여부 | 기능 활성화 상태 저장 |
| 정렬 설정 | 정렬 종류·오름/내림차순, 현재 적용된 정렬 상태 | 정렬 상태 유지 |
| 필터 설정 | 이용자가 직접 입력한 제외 키워드 | 검색결과 필터링 |
| 표시 설정 | 검색 개수 고정값, 광고요소 프리셋 해제 목록 | 화면 표시 유지 |

- 위 항목은 특정 개인을 식별할 수 있는 정보가 아니며, 이용자가 확장 설정에서 직접 입력·변경한 값입니다.
- 「제외 키워드」는 이용자가 입력한 검색 필터 문자열로, 이용자 브라우저에만 저장되고 외부로 전송되지 않습니다.

## 3. 저장 위치 및 보유·파기 (개인정보 보호법 제21조 참조)

- 위 설정값은 브라우저의 `chrome.storage.sync` 및 `chrome.storage.local`에 저장됩니다.
- `chrome.storage.sync`에 저장된 값은 이용자가 브라우저에 로그인한 본인 계정을 통해 브라우저(Google Chrome) 자체 기능으로 이용자의 다른 기기와 동기화될 수 있습니다. 이 동기화는 브라우저 제공사의 기능이며, 본 확장 개발자는 해당 데이터에 접근할 수 없습니다.
- 보유기간: 이용자가 확장을 사용하는 동안 유지됩니다.
- 파기: 이용자가 **확장 프로그램을 삭제하거나, 설정을 초기화(전체 해제)**하면 저장된 값은 지체 없이 삭제됩니다. 브라우저의 확장/사이트 데이터 삭제 기능으로도 제거할 수 있습니다.

## 4. 제3자 제공 및 처리위탁

본 확장은 저장된 설정값을 **제3자에게 제공하거나 처리를 위탁하지 않습니다.** 광고·분석(애널리틱스)·추적 목적의 수집이나 전송은 일절 수행하지 않습니다.

## 5. 개인정보 자동 수집 장치의 설치·운영

본 확장은 쿠키, 웹비콘 등 개인정보를 자동으로 수집하는 장치를 설치·운영하지 않습니다. 설정값 저장에는 브라우저 확장 저장소(`chrome.storage`)만 사용합니다.

## 6. 정보주체의 권리와 행사 방법 (개인정보 보호법 제4조 참조)

이용자는 언제든지 다음의 방법으로 저장된 설정값을 직접 관리할 수 있습니다.

- 열람·정정: 확장 설정 팝업에서 현재 설정을 확인·변경
- 삭제·파기: 설정 초기화, 확장 프로그램 삭제, 또는 브라우저의 사이트/확장 데이터 삭제
- 본 확장은 별도의 서버 저장이 없으므로, 위 브라우저 내 조치만으로 관련 데이터가 완전히 제거됩니다.

## 7. 안전성 확보 조치

- 설정값은 외부로 전송되지 않고 이용자 브라우저 내에만 저장됩니다.
- 본 확장은 이용자의 개인정보를 수집·보관하는 서버가 없어, 외부 유출 위험이 원칙적으로 존재하지 않습니다.
