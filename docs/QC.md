# QC 체크리스트

[`버전.md`](버전.md)·[`업그레이드.md`](업그레이드.md)·[`방법론.md`](방법론.md)에 **고쳐 두었다고 적힌 오류·동작**이 다시 깨지지 않았는지, **현재 기능이 기대대로 도는지** 확인한다.
