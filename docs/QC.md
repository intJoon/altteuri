## 재사용 항목 (다음 릴리스)

### 버전·스키마

- [ ] manifest / 버전.md / 방법론·업그레이드·출처 표기 일치
- [ ] `SETTINGS_VERSION` ↔ `settingsVersion` 일치
- [ ] 스키마·키 변경이면 업그레이드.md에 마이그레이션 기록

### 자동

- [ ] `node --check` (변경된 extension·web JS)
- [ ] `node --test tests/extension/*.mjs`
- [ ] 마스터 스위치·금지 문구·`softwareVersion` 잔존 검색

### 수동 (쿠팡)

- [ ] 설치 경로·팝업·opt-in 기본
- [ ] 기능별 on/off
- [ ] 정렬 순번·담기 성공 판정·의견 제출/조회
- [ ] DOM 불확실 시 추정하지 않는지

## 자동 검증 명령

```bash
node --test tests/extension/*.mjs
```

PowerShell에서 버전·잔존 검색·구문 검사는 저장소 루트에서 `docs/QC.md` 갱신 직전 스크립트로 돌린다.
