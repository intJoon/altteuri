# 버전

`manifest.json`과 맞추는 SemVer 릴리스 기록이다. 각 항목은 **그 버전에서 일어난 변화**를 설명하며, 폐기된 기능은 후속 버전과 [`업그레이드.md`](업그레이드.md)를 함께 봐야 한다.

## 1.16.4

- 의견 제출 전에 본문·확장 버전의 저장·공개 가능성, 미국 소재 서비스 처리, 개인정보 입력 금지를 안내한다.
- 개인정보처리방침에서 목록 조회(GET)와 제출(POST)을 구분하고, 의견 최대 2년·요청 제한 식별값 7일의 보존 정책을 명시했다. 의견 API는 요청 시 730일이 지난 의견을 삭제한다.
- 개인정보처리방침과 이용약관을 `legal.html`의 원문으로 사용하는 생성 절차를 도입했다.
- 정렬 순위 표시를 화면에 보이는 상품만 대상으로 하고, 표시 항목 숨김 뒤 정렬을 다시 적용하도록 보완했다.
- 팝업에서 의견 보내기 진입을 제목 바로 옮기고 의견 화면 높이를 맞췄다.

## 1.16.3

- 확장 팝업과 소개 웹의 의견 시각에서 시·분을 제거하고 날짜만 표시했다.

## 1.16.2

- 소개 웹의 의견 작성 폼을 제거했다. 이후 의견 제출은 확장 팝업에서만 가능하고 웹은 목록 조회만 제공한다.
- 등록 직후 페이지 offset, `더 보기` 중복 클릭, 창 크기·폰트 로딩 뒤 3줄 말줄임 계산을 보완했다.
- 제한 안내를 `하루에 2개까지 보낼 수 있습니다`로 통일했다.

## 1.16.1

- 의견 본문을 기본 3줄로 접고 필요한 경우에만 `펼치기`/`접기`를 표시했다.
- 팝업·웹·API의 기본 페이지 크기를 10건에서 5건으로 변경했다.

## 1.16.0

- 사용자 표시명을 `알뜰이`, 기능명을 `페이지 표시 항목 조정`으로 통일하고 아이콘·팝업·법적 고지의 시각 체계를 정리했다.
- 새 설치의 모든 기능을 opt-in으로 바꿨다. 기존의 같은 이름 설정은 기본값 병합으로 유지했다.
- 내부 `cra`/`addon` 접두사를 `alt`로 변경하고 설정 스키마를 8로 올렸다. 구 접두사 키의 자동 변환은 제공하지 않았다.
- 의견 작성·최근 목록을 확장 팝업과 소개 웹에 추가했다. 이 당시 웹에도 작성 기능이 있었으나 1.16.2에서 제거됐다.
- 의견은 최대 500자, 동일 IP의 제출은 UTC 하루 2건으로 제한했다. 원문 IP 대신 날짜별 HMAC 식별값과 횟수를 저장하도록 했다.

## 1.15.6

- 실제 장바구니 완료 UI의 `담기 완료`, `상품이 장바구니에 담겼습니다`, 장바구니 이동 버튼을 성공 판정에 추가했다.

## 1.15.5

- 헤더 장바구니 숫자와 타임아웃 성공 추정을 제거했다. 타임아웃은 실패로 처리했다.
- 장바구니 훅을 번들 파일 `page-cart-hook.js`의 MAIN world 주입으로 정리했다.

## 1.15.4

- 쿠팡 CSP로 인라인 장바구니 훅이 차단되는 문제에 대응하기 시작했다. 최종 주입 방식은 1.15.5에서 확정됐다.

## 1.15.3

- 1.15.2의 상품 카드 전체 호버·iframe 3개 캐시를 폐기했다.
- 담기 버튼에서만 160ms 워밍하고 iframe 하나를 사용하도록 축소했다. 담기 요청은 병렬 처리하고 완료 후 iframe을 폐기했다.

## 1.15.2

- 당시 장바구니 반응 속도를 줄이기 위해 상품 카드 호버 시 최대 3개의 상품 상세 iframe을 미리 준비했다. 이 방식은 1.15.3에서 폐기됐다.

## 1.15.1

- 정렬 순위, 장바구니 버튼, 법적 고지의 색·그림자·배지 스타일을 팝업 블루 중심으로 통일했다.

## 1.15.0

- 검색 결과에서 상품 상세를 숨김 iframe으로 열고 쿠팡의 네이티브 담기 버튼을 실행하는 `장바구니 바로 담기`를 추가했다.
- 스피너·성공·실패 상태만 표시하고 수량 조절이나 별도 토스트는 두지 않았다.
- 당시 새 설정의 기본값은 켜짐이었으나 1.16.0부터 새 설치 기본값은 꺼짐이다.

## 1.14.0

- `mc.coupang.com`을 콘텐츠 스크립트와 호스트 권한 범위에 추가했다.

## 1.13.1

## 1.13.0

- 개인정보처리방침과 이용약관, 최초 설치 법적 고지 화면을 추가했다.
- 이 버전에는 의견 서버 기능이 없었으므로 당시 고지는 브라우저 설정 저장만을 전제로 했다. 1.16.0 이후의 의견 처리는 현행 개인정보처리방침이 대체한다.
- 팝업 상세 화면의 뒤로가기와 제목 구성을 단순화했다.

## 1.12.2

- 적용 중인 정렬 종류와 검색어를 브라우저 local 저장소에 기록해 같은 검색어의 리로드에서 복원하도록 했다.
- 대부분의 팝업 설정을 탭 전체 리로드 대신 `storage.onChanged`로 반영했다. 활성 정렬 기능을 끄거나 마스터를 전환할 때만 필요한 리로드를 유지했다.
- 초기화 순서 때문에 콘텐츠 스크립트가 중단되던 문제를 수정했다.

- 단위가는 금액을 기준량으로 나눠 서로 다른 단위 표기를 비교하도록 했다.

- 쿠팡 정렬·부분 렌더링 뒤 키워드 숨김이 풀리던 문제를 수정했다.
